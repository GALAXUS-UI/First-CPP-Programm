About the project
-----------------

The first code i have written in C++.

Project goal
------------

The goal of this project was to learn the programming language C++, i used some knowledge i have from coding in c#.

Features
- A random number between 1 and 100 gets generated.
- The Player has 20 guesses before he loses.
- If he loses the Programm reveals the generated number.

What I Practiced
----------------

- using Random in C++
- Syntax in general.

How to download and execute
---------------

1. Go to the top right of the project and download the code in a ZIP file
2. Unzip the ZIP file
3. Check that Visual Studio 2026 is installed.
4. Open the folder and open the .xslnx file.
5. press F5 to execute the code.
